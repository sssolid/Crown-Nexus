app.core.celeryconfig
=====================

.. py:module:: app.core.celeryconfig

.. autoapi-nested-parse::

   Celery configuration settings.

   This module contains all Celery configuration settings including:
   - Broker and result backend
   - Task serialization
   - Worker settings
   - Task routing
