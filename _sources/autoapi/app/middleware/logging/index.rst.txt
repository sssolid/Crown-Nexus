.. py:class:: RequestLoggingMiddleware(app)

   Bases: :py:obj:`starlette.middleware.base.BaseHTTPMiddleware`


   Middleware for logging requests and setting up request context.

   This middleware generates a request ID for each incoming request, logs
   request and response information, and sets up the thread-local request
   context for use by other components.


   .. py:method:: dispatch(request, call_next)
