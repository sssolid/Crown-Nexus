app.core.dependency_manager.get_service(service_name, db=None)

Get a service by name.

Args:

service_name: Name of the service to retrieve db: Optional database session to pass to the service provider

Returns:

The service instance with the following type mappings: - “error_service” -> ErrorService - “user_service” -> UserService - “audit_service” -> AuditService - “search_service” -> SearchService - “media_service” -> MediaService

Examples:

```python # Get error service (returns ErrorService) error_service = get_service(“error_service”)

# Get user service with DB session (returns UserService) user_service = get_service(“user_service”, db=session) ```
