async app.db.utils.transaction(db)

Context manager for database transactions.

This creates a transaction context that automatically handles commits and rollbacks. If the session is already in a transaction, it will use the existing transaction.

Args:

db: The database session

Yields:

The database session

Raises:

TransactionException: If a database error occurs during the transaction

Parameters:

db (sqlalchemy.ext.asyncio.AsyncSession)

Return type:

AsyncGenerator[sqlalchemy.ext.asyncio.AsyncSession, None]
