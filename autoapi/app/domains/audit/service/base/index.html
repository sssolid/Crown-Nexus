async log_event(event_type, user_id=None, ip_address=None, resource_id=None, resource_type=None, details=None, context=None, level=None, options=None)

Log an audit event.

Args:

event_type: Type of the audit event user_id: ID of the user who performed the action ip_address: IP address of the user resource_id: ID of the resource affected resource_type: Type of the resource affected details: Additional details about the event context: Additional context information level: Severity level of the event options: Audit logging options

Returns:

str: The ID of the created audit event
