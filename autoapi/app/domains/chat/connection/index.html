app.domains.chat.connection

WebSocket connection management.

This module manages WebSocket connections for the real-time chat system: - Tracking active connections - Sending messages to specific connections - Broadcasting to multiple connections - Connection authentication and validation

It provides the foundation for real-time communication in the chat system.
