app.domains.chat.tasks

Celery worker for background tasks.

This module configures the Celery worker for asynchronous background tasks: - Message notifications - Chat analytics - Content moderation - Scheduled operations

These background tasks help offload CPU-intensive or time-consuming operations from the main application thread.
