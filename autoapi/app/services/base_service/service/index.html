class app.services.base_service.service.BaseService(db, model_class, create_schema, update_schema, response_schema, repository_class=BaseRepository)

Bases: Generic[T, C, U, R, ID], app.services.base_service.contracts.BaseServiceProtocol[T, ID, C, U, R]

Base service for CRUD operations on entities.

This service provides standardized CRUD operations with: - Integrated permissions checking - Transaction management - Error handling - Validation - Event dispatching - Logging
